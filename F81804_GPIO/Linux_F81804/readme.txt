Linux 版 F81804_GPIO


1、sudo apt-get install gcc (ubuntu) 	[Centos] ( yum install gcc )

2、gcc -o F81804_GPIO F81804_GPIO.c

3、su (以 root 身份运行)

4、./F81804_GPIO


---------------------------------------------------------------------------------------   操作说明：---------------------------------------------------------------------------------------

- 根据提示控制 GPIO。

- 例如 控制 GPIO 50 即 Pin1，

- 在程序中输入 1,0 （控制GPIO 1脚 转换为输出模式，值为0） 

- 在程序中输入 1,1 （控制GPIO 1脚 转换为输出模式，值为1） 

- 在程序中输入 rd1（控制GPIO 1脚 转换为输入模式）
